

## Add Apple Touch Icon

**File: `index.html`** — Add an `apple-touch-icon` link tag in the `<head>`, reusing the existing `favicon.png`:

```html
<link rel="apple-touch-icon" href="/favicon.png" />
```

This resolves the mobile optimization warning. For best results, the icon should ideally be 180×180px. If your current `favicon.png` is smaller, consider uploading a higher-resolution version.
